## Run Locally

Clone the project

```bash
  git clone https://github.com/abhishek-dagar/Algorithm-Visualizer.git
```


Go to the project directory

```bash
  cd Algorithm-Visualizer
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  yarn dev
```
